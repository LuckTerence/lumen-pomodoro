# 项目记忆

## 项目概况
- **项目名称**: Lumen Pomodoro（摄像头提醒番茄钟）
- **技术栈**: .NET 8 + WPF + WPF-UI + Windows Media Foundation
- **架构**: MVVM + DI（构造函数注入）
- **数据存储**: 本地 JSON（`%APPDATA%/LumenPomodoro/`）

## 关键约定
- 不自动进入休息、不自动循环番茄钟 — 所有关键流程由用户手动确认
- 摄像头仅用于触发硬件指示灯和走神检测，不拍照/不录像/不上传/不显示画面
- 首次使用摄像头时弹隐私声明，需用户明确同意
- 发布包：小启动器（检测 Runtime）+ 主程序，控制在约 16MB
