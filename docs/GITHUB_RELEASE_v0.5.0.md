# Lumen Pomodoro v0.5.0

**灵动岛优先** · 本地优先 · Windows + macOS

专注时几乎不用开主窗口——屏幕上方的胶囊就是番茄钟。

## 下载

| 平台 | 文件 |
|------|------|
| **Windows x64** | `LumenPomodoro.zip` |
| **macOS 14+** | `LumenPomodoro-mac-0.5.0.dmg` |

## 本版变化

- **定位切换**：默认卖点从摄像头灯 → **灵动岛主交互**（灯为可选）  
- **三态**：Compact / Expanded / Transient  
- **岛上操作**：暂停、继续、结束休息、开始、打开主窗  
- **主窗前台策略**：淡化 / 保持 / 隐藏（可设）  
- **完成 / 预告 / 走神** 优先走 Transient  
- 引导与场景预设文案同步更新  

## 演示路径

开始专注 → 切到其它窗口 → 顶部岛走时 → 点击岛暂停。

## 隐私

本地数据；默认不调用摄像头。开启灯时仍不拍照、不录像、不上传。
