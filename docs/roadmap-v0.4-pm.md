## 验收：核心路径

1. 首次启动 → 引导 → 选场景 → 进入主界面  
2. 选任务 → 专注 →（可选灯）→ 完成 → 休息（可选全屏/严格）→ 统计可见  
3. 专注中退出需确认；勿扰不刷屏；预告一次
