# 核心路径验收清单（发版必过）

> 产品定位（方案 B）：**切窗口也不丢——顶栏胶囊随时控番茄**。  
> 摄像头灯为高级可选，默认关，不进入主路径必过项。

双端（Windows / macOS）各跑一遍。不通过不发 tag。

## A. 首次体验

| # | 步骤 | 期望 |
|---|------|------|
| A1 | 全新数据目录启动 | 出现首次引导（非直接进日报） |
| A2 | 引导：岛说明 → 隐私 → 选「标准」 | 灵动岛开启；摄像头灯**关**；防走神开 |
| A3 | 设置 → 重新查看首次引导 | 可再次完成引导 |
| A4 | 引导选「轻松」 | 岛开、摄像头关、防走神关 |
| A5 | 设置页结构 | 顶部有「灵动岛」；摄像头在底部「高级」 |

## B. 专注主路径（岛）

| # | 步骤 | 期望 |
|---|------|------|
| B1 | 选任务 → 开始专注 | 倒计时运行；顶栏岛显示剩余时间 |
| B2 | 切到其它窗口 | 岛仍可见并继续走时 |
| B3 | 点岛暂停 / 继续 | 状态正确；Mac 菜单栏同步 |
| B4 | 空格暂停/继续（主窗） | 与岛状态一致 |
| B5 | 专注完成 | 评分/休息可选；岛 Transient 或完成面板 |
| B6 | 短休 / 长休 | 倒计时；全屏休息若开启则遮罩 |
| B7 | 严格模式结束休息 | 不可提前结束；提示严格模式 |
| B8 | 统计页 | 可见今日/热力或趋势数据 |

## C. 边界与安全

| # | 步骤 | 期望 |
|---|------|------|
| C1 | 专注中关应用 | 确认对话框；取消则继续 |
| C2 | 结束前预告（默认 30s） | 只提醒一次 |
| C3 | 走神黑名单 | 防抖后提醒，同会话有上限 |
| C4 | 勿扰/锁屏（若可测） | 不刷屏通知 |
| C5 | 数据目录 | 有 settings/tasks/sessions JSON |

## D. 高级 · 摄像头灯（可选，非阻断发版）

| # | 步骤 | 期望 |
|---|------|------|
| D1 | 设置 → 高级 开启灯 | 状态文案变为待命 |
| D2 | 专注完成 | 灯亮；状态「亮着」 |
| D3 | 拒摄像头权限后开灯 | 错误提示 + 可打开系统设置 |

## E. 打包

| # | 步骤 | 期望 |
|---|------|------|
| E1 | Windows zip | 根目录启动器 + `app/` 主程序可运行 |
| E2 | Mac dmg | 可拖入应用程序并启动 |
| E3 | Release CI | tag `v*` 产出双端产物 |
