# 核心路径验收清单（发版必过）

双端（Windows / macOS）各跑一遍。不通过不发 tag。

## A. 首次体验

| # | 步骤 | 期望 |
|---|------|------|
| A1 | 全新数据目录启动 | 出现首次引导（非直接进日报） |
| A2 | 引导：灯说明 → 隐私 → 选「标准」 | 摄像头提醒开启，引导关闭 |
| A3 | 设置 → 重新查看首次引导 | 可再次完成引导 |
| A4 | 引导选「轻松」 | 摄像头关、防走神关 |

## B. 专注主路径

| # | 步骤 | 期望 |
|---|------|------|
| B1 | 选任务 → 开始专注 | 倒计时运行；防走神按设置生效 |
| B2 | 空格暂停/继续 | 状态正确；Mac 菜单栏同步 |
| B3 | 专注完成 | 评分/休息可选；灯（若开启）亮起 |
| B4 | 灯状态文案 | 显示「亮着/待命/关闭/异常」之一 |
| B5 | 短休 / 长休 | 倒计时；全屏休息若开启则遮罩 |
| B6 | 严格模式结束休息 | 不可提前结束；提示严格模式 |
| B7 | 统计页 | 可见今日/热力或趋势数据 |

## C. 边界与安全

| # | 步骤 | 期望 |
|---|------|------|
| C1 | 专注中关应用 | 确认对话框；取消则继续 |
| C2 | 结束前预告（默认 30s） | 只提醒一次 |
| C3 | 走神黑名单 | 防抖后提醒，同会话有上限 |
| C4 | 勿扰/锁屏（若可测） | 不刷屏通知 |
| C5 | 拒摄像头权限后开灯 | 错误提示 + 可打开系统设置 |
| C6 | 数据目录 | 有 settings/tasks/sessions JSON |

## D. 打包

| # | 步骤 | 期望 |
|---|------|------|
| D1 | Windows zip | 根目录启动器 + `app/` 主程序可运行 |
| D2 | Mac dmg | 可拖入应用程序并启动 |
| D3 | Release CI | tag `v*` 产出双端产物 |
