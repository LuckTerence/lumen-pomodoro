# Lumen Pomodoro v0.4.0

**摄像头指示灯番茄钟** · 本地优先 · Windows + macOS

专注结束点亮硬件指示灯，备考时也能被提醒休息——不拍照、不录像、不上传。

## 下载

| 平台 | 文件 | 说明 |
|------|------|------|
| **Windows x64** | `LumenPomodoro.zip` | 解压后运行根目录 `LumenPomodoro.exe` |
| **macOS 14+** | `LumenPomodoro-mac-0.4.0.dmg` | 拖到「应用程序」；若拦截请右键打开 |

## Windows 运行

1. 解压 zip  
2. 双击 `LumenPomodoro.exe`（启动器）  
3. 若提示安装 .NET Desktop Runtime，按提示完成即可  
4. 首次启动完成引导：为什么用灯 → 隐私 → 选场景（轻松 / 标准 / 严格）

## macOS 运行

1. 打开 dmg，拖入应用程序  
2. 首次引导中可按需授权：摄像头 / 通知 / 辅助功能  
3. 菜单栏可直接开始/暂停专注，并显示剩余时间  

## 本版亮点

- **首次引导**与**三场景预设**（轻松 / 标准 / 严格专注）  
- **摄像头灯**可读状态；失败时可打开系统隐私设置  
- **FocusGuard** 防抖、每会话提醒上限、遵从勿扰  
- **全屏休息**、**严格模式**（不可手关灯、不可提前结束休息）  
- **结束前预告**、**计时中退出确认**  
- 数据仅本机；无账号、无云同步  

## 隐私

- 不保存照片 · 不录制视频 · 不上传画面  
- 摄像头仅用于点亮本机指示灯
