# 灵动岛优先 · 产品规格（v0.5）

> 北极星：专注时几乎不用打开主窗口——**屏幕上方的岛就是番茄钟**。  
> 摄像头灯降为可选增强（设置中开启）。

## 三态

| 态 | 用途 |
|----|------|
| **Compact** | 默认：模式文案 + 倒计时；点击展开 |
| **Expanded** | 操作：暂停/继续、结束休息、开始专注、打开主窗；**Idle 可选任务芯片**；约 5s 无操作收回 |
| **Transient** | 完成 / 预告 / 走神等事件，弹性弹出，约 2.6s 后回到 Compact（若仍在计时）或收起 |

### 岛上选任务

- **Idle / 待命岛**：失焦时显示「准备 · 任务名」+ 时长；展开后横向任务芯片（最多 8）
- **仅 Idle 可切换任务**（专注中只读当前任务色点）
- 选中后点「开始」直接开专注

## 主窗前台策略 `DynamicIslandWhenFocused`

| 值 | 行为 |
|----|------|
| `keep` | 前台也完整显示 |
| `minimize`（默认） | 前台淡化缩小，仍可见 |
| `hide` | 前台隐藏，失焦再出现 |

## 场景预设

| 场景 | 岛 | 灯 | 其它 |
|------|----|----|------|
| 轻松 | 开 / minimize | 关 | 无防走神 |
| 标准 | 开 / minimize | 关 | 防走神 |
| 严格 | 开 / keep | 关 | 全屏休息 + 严格 |

## 演示路径（15 秒）

开始专注 → 切到其它窗口 → 顶部岛继续走时 → 点岛可暂停。
